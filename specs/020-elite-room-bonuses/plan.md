# Implementation Plan: Elite Room Bonuses

**Branch**: `020-elite-room-bonuses` | **Date**: 2026-03-02 | **Spec**: [spec.md](spec.md)

## Summary

Add per-room-type enemy count and essence multipliers to the spawn config system.
EliteRoom01 gets `enemy_count_mult: 1.5` (floor) and `essence_mult: 1.8`. Both values
live in `dungeon_config.json`. `RoomSpawnConfig` parses them; `RoomSpawner` applies the
count multiplier at spawn time by cycling through existing spawn_points; `RunManager`
applies the essence multiplier per kill via a property read from the spawner.

## Technical Context

**Language/Version**: GDScript (Godot 4.6), static typing throughout
**Primary Dependencies**: RoomSpawner, RoomSpawnConfig, RunManager, ResourceManager (autoload)
**Storage**: `data/dungeon_config.json` — adds two optional fields to spawn_config entries; backward compatible (missing fields default to 1.0)
**Testing**: Manual validation in `quickstart.md` (10 scenarios)
**Target Platform**: Android mobile (portrait 1080×1920); Windows for development
**Performance Goals**: All methods O(1) or O(n) where n = spawn count (≤10). No per-frame cost.
**Constraints**: Extra enemies must cycle from existing spawn_points (no new config entries). Multipliers apply only to the room type that defines them — zero regression on standard rooms.

## Implementation Notes

### dungeon_config.json

```json
"EliteRoom01": {
    "enemy_count_mult": 1.5,
    "essence_mult": 1.8,
    "spawn_points": [
        { "enemy_id": "slime",    "position": { "x": -80, "y": 0 }, "radius": 20 },
        { "enemy_id": "skeleton", "position": { "x":  80, "y": 0 }, "radius": 20 }
    ]
}
```

### RoomSpawnConfig.gd

```gdscript
var enemy_count_mult: float = 1.0
var essence_mult: float = 1.0

# In from_dict():
cfg.enemy_count_mult = float(data.get("enemy_count_mult", 1.0))
cfg.essence_mult = float(data.get("essence_mult", 1.0))
```

### RoomSpawner.gd

```gdscript
# New computed property
var essence_mult: float:
    get: return _config.essence_mult if _config != null else 1.0

# Modified _spawn_enemies()
var base_count: int = _config.spawn_points.size()
_living_count = mini(floori(float(base_count) * _config.enemy_count_mult), MAX_ENEMIES)
if _living_count == 0:
    return
for i in _living_count:
    var sp: SpawnPointData = _config.spawn_points[i % base_count]
    # ... spawn using sp.enemy_id, sp.radius (unchanged mechanics)
```

### RunManager.gd

```gdscript
# Modified _on_enemy_defeated()
var room_essence_mult: float = (current_room as RoomSpawner).essence_mult if current_room != null else 1.0
var essence: int = floori(base_essence * (1.0 + essence_depth_scale * float(current_room_depth - 1)) * room_essence_mult)
```
