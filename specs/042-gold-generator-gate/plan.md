# Implementation Plan: Gold Generator Gate (Transmuter)

**Branch**: `042-gold-generator-gate` | **Date**: 2026-03-16 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/042-gold-generator-gate/spec.md`

---

## Summary

Gate gold accumulation (in-session and offline) behind a one-time Alchemy Lab upgrade called "Transmuter" (tech key `gold_generator`, cost 50 shards). Gold ticking and offline credit are suppressed until the upgrade is purchased. All values are data-driven via `meta_config.json`. The change touches seven files: one JSON config, three GDScript data/manager files, two scene-layer files (one script, one tscn Editor task).

---

- **III. Mobile-First** ✅ — Gold tick is a single float addition per `_process()` frame, now with one early-return guard prepended. No draw-call, shader, or physics impact.
- **IV. Editor-Centric** ✅ — One new `Button` node added to `LabUpgradeScreen.tscn` via Godot Editor. The `@export var _transmuter_button: Button` reference is assigned in the Inspector. No `.tscn` raw edits.
- **V. Simplicity & YAGNI** ✅ — No new abstractions, autoloads, or services. The feature fits in one new MetaState field, two guard clauses, one purchase method, and one LabUpgradeScreen button.
- **VI. Early Return** ✅ — Guards in `tick_gold()` and `apply_offline_gold()` are early returns. `_update_transmuter_button()` returns early on the "already owned" path. `purchase_gold_generator()` returns early on `already_owned` and `cannot_afford` paths.

No violations. **GATE PASSED.**

### Step 4 — Impl logic: `scripts/managers/MetaManager.gd` (MetaManagerImpl)

**4a. Guard `tick_gold()`** — prepend early return:
```gdscript
func tick_gold(delta: float, rate_per_hour: float) -> int:
    if not meta_state.gold_generator_owned:
        return floori(meta_state.total_gold)
    meta_state.total_gold += delta * rate_per_hour / 3600.0
    return floori(meta_state.total_gold)
```

**4b. Guard `apply_offline_gold()`** — prepend early return:
```gdscript
func apply_offline_gold(now_unix: int, rate_per_hour: float, save_manager: Node) -> void:
    if not meta_state.gold_generator_owned:
        return
    if meta_state.gold_last_saved_timestamp == 0:
        return
    var elapsed: int = now_unix - meta_state.gold_last_saved_timestamp
    if elapsed <= 0:
        return
    meta_state.total_gold += float(elapsed) * rate_per_hour / 3600.0
    _save(save_manager)
```

**4c. New method `purchase_gold_generator()`**:
```gdscript
## Purchases the Transmuter (gold generator) if affordable and not already owned. Returns true on success.
func purchase_gold_generator(cost: int, save_manager: Node) -> bool:
    if meta_state.gold_generator_owned:
        return false
    if not can_spend(cost):
        return false
    meta_state.total_shards -= cost
    meta_state.gold_generator_owned = true
    _save(save_manager)
    return true
```

func purchase_gold_generator() -> bool:
    var cost: int = ResourceManager.get_meta_config().get("alchemy_lab", {}).get("upgrades", {}).get("gold_generator", {}).get("cost", 50)
    var success: bool = _impl.purchase_gold_generator(cost, SaveManager)

        shards_changed.emit(meta_state.total_shards)
    return success

### Step 6 — UI script: `scenes/hub/LabUpgradeScreen.gd`

Add export:
```gdscript
@export var _transmuter_button: Button
```

Wire in `_ready()`:
```gdscript
_transmuter_button.pressed.connect(_on_transmuter_pressed)
```

Refactor `_update_buttons()` to call two helpers (extract existing essence logic, add transmuter logic):
```gdscript
func _update_buttons() -> void:
    _update_essence_button()
    _update_transmuter_button()


func _update_essence_button() -> void:
    var upgrade: Dictionary = ResourceManager.get_meta_config().get("alchemy_lab", {}).get("upgrades", {}).get("essence_gain", {})
    var upgrade_name: String = upgrade.get("name", "essence_gain")
    var max_levels: int = upgrade.get("max_levels", 1)
    var level: int = MetaManager.meta_state.essence_gain_level
    if level >= max_levels:
        _essence_button.text = "{n} — MAX".format({"n": upgrade_name})
        _essence_button.disabled = true
        return
    var base_cost: int = upgrade.get("base_cost", 0)
    var pct: int = int(float(level + 1) * upgrade.get("essence_per_level", 0.05) * 100.0)
    _essence_button.text = "{n} +{pct}% (Lv{lv})".format({"n": upgrade_name, "pct": pct, "lv": level + 1})
    _essence_button.disabled = base_cost == 0 or not MetaManager.can_spend(base_cost)


func _update_transmuter_button() -> void:
    var upgrade: Dictionary = ResourceManager.get_meta_config().get("alchemy_lab", {}).get("upgrades", {}).get("gold_generator", {})
    var upgrade_name: String = upgrade.get("name", "gold_generator")
    var cost: int = upgrade.get("cost", 50)
    if MetaManager.is_gold_generator_owned:
        _transmuter_button.text = "{n} — ACTIVE".format({"n": upgrade_name})
        _transmuter_button.disabled = true
        return
    _transmuter_button.text = "{n} ({cost} shards)".format({"n": upgrade_name, "cost": cost})
    _transmuter_button.disabled = not MetaManager.can_spend(cost)


func _on_transmuter_pressed() -> void:
    MetaManager.purchase_gold_generator()
```

### Step 8 — Unit tests

Extend or add unit tests covering:
- `tick_gold()` returns current floor without mutating `total_gold` when `gold_generator_owned == false`
- `apply_offline_gold()` awards 0 when `gold_generator_owned == false`, even with valid timestamp
- `purchase_gold_generator()` succeeds with sufficient shards, sets flag, deducts shards
- `purchase_gold_generator()` fails with insufficient shards
- `purchase_gold_generator()` is idempotent (second call returns false, no double-deduct)
- After purchase: `tick_gold()` increments `total_gold` normally
- After purchase: `apply_offline_gold()` credits offline time normally
