# Implementation Plan: Magic Forge ‚Äî Missile Extra Charge Upgrade

**Branch**: `066-missile-extra-charge` | **Date**: 2026-03-20 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/066-missile-extra-charge/spec.md`

## Summary

Add a one-time Magic Forge purchase ("Arcane Reservoir", 150 shards) that permanently grants Magic Missile +1 max charge. The upgrade is stored in `MetaState.missile_extra_charge_owned`, persisted in the save file, applied in `SkillComponent._load_skill_data()`, and surfaced via a new button in `ForgeUpgradeScreen`.

## Technical Context

**Language/Version**: GDScript (Godot 4.6), static typing throughout
**Primary Dependencies**: Existing autoloads ‚Äî `MetaManager`, `ResourceManager`, `SaveManager`; existing scripts ‚Äî `SkillComponent`, `ForgeUpgradeScreen`
**Storage**: `user://meta_save.json` (extended with one new boolean key)
**Testing**: GUT unit tests in `tests/unit/`
**Target Platform**: Android mobile portrait 1080√ó1920; Windows for development
**Performance Goals**: 60 fps unchanged; this feature adds one boolean property read at run start ‚Äî negligible cost
**Constraints**: Mobile renderer; no `.tscn` edits as raw text (Editor-only)
**Scale/Scope**: Single boolean field + one UI button

### Step 4 ‚Äî MetaManagerImpl purchase method

**File**: `scripts/managers/MetaManagerImpl.gd`

```gdscript
func purchase_missile_extra_charge(cost: int, save_manager: Node) -> bool:
	if meta_state.missile_extra_charge_owned:
		return false
	if not can_spend(cost):
		return false
	meta_state.total_shards -= cost
	meta_state.missile_extra_charge_owned = true
	_save(save_manager)
	return true
```

---

### Step 5 ‚Äî MetaManager autoload

**File**: `autoload/MetaManager.gd`

Add computed property:
```gdscript
var is_missile_extra_charge_owned: bool:
	get:
		return _impl.meta_state.missile_extra_charge_owned
```

Add delegating method:
```gdscript
func purchase_missile_extra_charge() -> bool:
	var cfg: Dictionary = ResourceManager.get_meta_config() \
		.get("magic_forge", {}).get("upgrades", {}) \
		.get("missile_charge_upgrade", {})
	var cost: int = cfg.get("cost", 150)
	var success: bool = _impl.purchase_missile_extra_charge(cost, SaveManager)
	if success:
		shards_changed.emit(meta_state.total_shards)
	return success
```

---

### Step 6 ‚Äî SkillComponent charge application

**File**: `scenes/player/components/SkillComponent.gd`

Inside `_load_skill_data()`, immediately after the line `_max_charges = int(...)` and before `_current_charges = _max_charges`:

```gdscript
if MetaManager.is_missile_extra_charge_owned:
    _max_charges += 1
```

No other changes needed; `_current_charges = _max_charges` and `_reset_charges()` both already use `_max_charges`.

### Step 7 ‚Äî ForgeUpgradeScreen UI logic

**File**: `scenes/hub/ForgeUpgradeScreen.gd`

Add export:
```gdscript
@export var _missile_charge_button: Button
```

In `_ready()`, connect the new button:
```gdscript
_missile_charge_button.pressed.connect(_on_missile_charge_button_pressed)
```

Also connect `MetaManager.shards_changed` to `_update_buttons` if not already (check existing code ‚Äî damage upgrade may already do this).

Add to `_update_buttons()`:
```gdscript
var mc_cfg: Dictionary = ResourceManager.get_meta_config() \
    .get("magic_forge", {}).get("upgrades", {}) \
    .get("missile_charge_upgrade", {})
var mc_name: String = mc_cfg.get("name", "Arcane Reservoir")
var mc_cost: int = mc_cfg.get("cost", 150)

if MetaManager.is_missile_extra_charge_owned:
    _missile_charge_button.text = "{n} ‚Äî Purchased".format({"n": mc_name})
    _missile_charge_button.disabled = true
elif MetaManager.can_spend(mc_cost):
    _missile_charge_button.text = "{n} ‚Äî {c} shards".format({"n": mc_name, "c": mc_cost})
    _missile_charge_button.disabled = false
else:
    _missile_charge_button.text = "{n} ‚Äî {c} shards (insufficient)".format({"n": mc_name, "c": mc_cost})
    _missile_charge_button.disabled = true
```

Add handler:
```gdscript
func _on_missile_charge_button_pressed() -> void:
	MetaManager.purchase_missile_extra_charge()
	_update_buttons()
```
