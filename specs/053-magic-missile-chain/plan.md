# Implementation Plan: Magic Missile Chain Relic

**Branch**: `053-magic-missile-chain` | **Date**: 2026-03-17 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/053-magic-missile-chain/spec.md`

## Summary

Add an uncommon relic (`chaining_stone`) that causes magic missile to strike a second enemy for a configurable fraction of primary damage (`chain_damage_mult = 0.5`, stored in `data/skills.json`). On projectile impact, if the relic is held, the closest living enemy to the hit point (excluding the primary target) receives `_damage × _chain_damage_mult`. All config lives in JSON; no balance constants in code.

### Step 6 — `Projectile.gd`

1. Add instance variable:
   ```gdscript
   var _chain_damage_mult: float = 1.0
   ```

2. Extend `setup()` signature and body:
   ```gdscript
   func setup(target: Enemy, damage: float, speed: float, max_distance: float, chain_damage_mult: float) -> void:
       _target = target
       _damage = damage
       _speed = speed
       _max_distance = max_distance
       _chain_damage_mult = chain_damage_mult
       _hit_area.body_entered.connect(_on_body_entered)
   ```

3. Update `_on_body_entered()`:
   ```gdscript
   func _on_body_entered(body: Node2D) -> void:
       if not body is Enemy:
           return
       var primary: Enemy = body as Enemy
       primary.take_damage(_damage)
       _try_chain(primary)
       queue_free()
   ```

4. Add `_try_chain()`:
   ```gdscript
   func _try_chain(primary_target: Enemy) -> void:
       if not RelicManager.has_chain_relic():
           return
       if RunManager.current_room == null:
           return
       var room_node: Node = RunManager.current_room.get_parent()
       var chain_target: Enemy = null
       var closest_dist: float = INF
       for child: Node in room_node.get_children():
           if not child is Enemy:
               continue
           if child == primary_target:
               continue
           if not is_instance_valid(child):
               continue
           var dist: float = global_position.distance_to((child as Enemy).global_position)
           if dist >= closest_dist:
               continue
           chain_target = child as Enemy
           closest_dist = dist
       if chain_target == null:
           return
       chain_target.take_damage(_damage * _chain_damage_mult)
   ```
