# Implementation Plan: Homing Projectile Skill

**Branch**: `046-homing-projectile-skill` | **Date**: 2026-03-17 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/046-homing-projectile-skill/spec.md`

## Summary

Add a skill button to the ExplorationHUD that fires a homing projectile toward the closest living enemy. The projectile continuously steers toward its target, deals 50% of the player's current attack damage on impact, and self-destructs on hit, target death, or exceeding a maximum travel distance. Multiple projectiles may coexist. No cooldown in this iteration.

## Technical Context

**Language/Version**: GDScript (Godot 4.6 — static typing required)
**Primary Dependencies**: Godot engine — `Area2D`, `CharacterBody2D`, `PhysicsProcess`, `GlobalSignals` (existing autoload)
**Storage**: `data/skills.json` — new `homing_projectile` entry (speed, max_distance)
**Testing**: Manual playtest via Godot editor + DevPanel; GUT unit tests if applicable
**Target Platform**: Android mobile (portrait 1080×1920); Windows for development
**Project Type**: Single Godot project
**Performance Goals**: Sustain 60 fps; projectiles are simple Node2D + Area2D nodes — negligible cost
**Constraints**: Mobile renderer only; no Forward+ effects; no new autoloads; Jolt physics
**Scale/Scope**: Up to several simultaneous projectiles per room; no pool needed at this scale

### Step 4 — Projectile: `scenes/combat/projectiles/Projectile.gd`

```gdscript
class_name Projectile
extends Node2D

var _target: Enemy
var _damage: float
var _speed: float
var _max_distance: float
var _distance_traveled: float = 0.0

@export var _hit_area: Area2D

func setup(target: Enemy, damage: float, speed: float, max_distance: float) -> void:
    _target = target
    _damage = damage
    _speed = speed
    _max_distance = max_distance
    _hit_area.body_entered.connect(_on_body_entered)

func _physics_process(delta: float) -> void:
    if not is_instance_valid(_target):
        queue_free()
        return
    var direction: Vector2 = global_position.direction_to(_target.global_position)
    var step: float = _speed * delta
    global_position += direction * step
    _distance_traveled += step
    if _distance_traveled >= _max_distance:
        queue_free()

func _on_body_entered(body: Node2D) -> void:
    if not body is Enemy:
        return
    (body as Enemy).take_damage(_damage)
    queue_free()
```

**Scene**: `Projectile.tscn` — root `Node2D` with:
- `Area2D` (`_hit_area`) — exported, with `CollisionShape2D` (small 12×12 rect or circle); collision layer set to detect enemy physics bodies.
- `ColorRect` — 16×16 px, yellow (`Color(1, 0.9, 0)`), offset `(-8, -8)` to center on root.

### Step 5 — Skill: `scenes/player/components/SkillComponent.gd`

```gdscript
class_name SkillComponent
extends Node

const PROJECTILE_SCENE: PackedScene = preload("res://scenes/combat/projectiles/Projectile.tscn")
const SKILL_ID: String = "homing_projectile"

@export var _combat_component: CombatComponent

var _speed: float = 600.0
var _max_distance: float = 2200.0

func _ready() -> void:
    _load_skill_data()
    GlobalSignals.skill_button_pressed.connect(_on_skill_button_pressed)

func _load_skill_data() -> void:
    var skills: Array = ResourceManager.get_skills()
    for entry: Variant in skills:
        if not entry is Dictionary:
            continue
        if entry.get("id", "") != SKILL_ID:
            continue
        _speed = entry.get("speed", _speed)
        _max_distance = entry.get("max_distance", _max_distance)
        return

func _on_skill_button_pressed() -> void:
    if not RunManager.is_run_active:
        return
    if RunManager.current_room == null:
        return
    var target: Enemy = _find_closest_enemy()
    if target == null:
        return
    var damage: float = floorf(_combat_component.attack_damage * 0.5)
    var room_node: Node = RunManager.current_room.get_parent()
    var projectile: Projectile = PROJECTILE_SCENE.instantiate()
    room_node.add_child(projectile)
    projectile.global_position = global_position
    projectile.setup(target, damage, _speed, _max_distance)

func _find_closest_enemy() -> Enemy:
    if RunManager.current_room == null:
        return null
    var room_node: Node = RunManager.current_room.get_parent()
    var closest: Enemy = null
    var closest_dist: float = INF
    for child: Node in room_node.get_children():
        if not child is Enemy:
            continue
        if not is_instance_valid(child):
            continue
        var dist: float = global_position.distance_to((child as Enemy).global_position)
        if dist >= closest_dist:
            continue
        closest = child as Enemy
        closest_dist = dist
    return closest
```

**Inspector assignments** (done in Godot Editor, inside `Player.tscn`):
- `_combat_component` → drag `CombatComponent` node from scene tree

### Collision Layers

The projectile's `Area2D` must be on a physics layer that overlaps with the enemy's `CharacterBody2D` layer. Verify in Project Settings → Physics Layers that the enemy body layer and projectile area mask are compatible. Use the same layer/mask settings as the existing `AttackArea` on the player (CombatComponent's `_attack_area`).
